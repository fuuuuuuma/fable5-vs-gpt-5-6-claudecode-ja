# 🎁 料金・コスパ早見表＆ざっくり計算ガイド

「Fable 5 と GPT-5.6、自分の使い方だと月いくらくらい？」を、専門知識なしでざっくり掴むためのシートです。難しい計算は不要。考え方の型だけ持ち帰ってください。

**受け取り方**: 下のコードブロック右上のコピーボタンでコピーするか、このファイルをそのままダウンロードしてください。

**使い方**: まず料金表で単価の差を掴み、「ざっくり月額の見積もり方」の3ステップで自分の規模に当てはめます。

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 API 料金（100万トークンあたり）※2026年7月時点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 モデル            入力     出力
 Claude Fable 5     $10     $50    ← 最上位ゆえ高い
 GPT-5.6 Sol        $5      $30    ← Fable 5 の約半額
 GPT-5.6 Terra      $2.50   $15    ← 日常運用向け
 GPT-5.6 Luna       $1      $6     ← 大量処理向け


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 トークンってどれくらい？（ざっくりの感覚）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
・日本語 約1文字 ≒ 1〜2トークン
・短いチャット1往復 ≒ 数百〜数千トークン
・長い資料1本の要約 ≒ 数千〜数万トークン
 ※「入力」は渡す文章、「出力」は返ってくる文章。
   長い返答ほど「出力」の課金が効く（Fable 5 は出力が高い）。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 ざっくり月額の見積もり方（3ステップ）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1) 1日に何回くらい頼む？（例：20回）
2) 1回の“重さ”を選ぶ
     軽い（質問1つ）   ≒  2,000 トークン
     ふつう（下書き）  ≒  8,000 トークン
     重い（長文・資料） ≒ 30,000 トークン
3) 回数 × 重さ × 30日 × 単価 で、だいたいの月額が出る
     → 出力が長いほど跳ねる。だから Fable 5 は
       「難所だけ」に絞ると、ぐっと安くなる。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 コスパの結論
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
・日常づかい          → GPT-5.6（同じ賢さを実費 約1/3）
・精度が命の一発        → Fable 5（難所に絞る）
・重い作業をたくさん      → Terra/Luna で回し、要所だけ Sol/Fable 5

 ※ ChatGPT や Claude の「会話アプリ」は月額プラン制です。
   上の従量課金は、API を直接使う場合の目安。ふだんの
   チャット利用なら、まず各社の無料枠・有料プランで十分です。
```

**ポイント**: 「安いモデルで8割、高いモデルで残り2割」。これだけで、精度を落とさずに月のコストは大きく下がります。数字はすべて2026年7月時点の目安で、最新は各公式の料金ページで確認してください。
