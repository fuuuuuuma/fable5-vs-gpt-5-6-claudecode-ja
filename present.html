   長い返答ほど「出力」の課金が効く（Fable 5 は出力が高い）。

 ざっくり月額の見積もり方（3ステップ）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2) 1回の“重さ”を選ぶ
     軽い（質問1つ）   ≒  2,000 トークン
     ふつう（下書き）  ≒  8,000 トークン
     重い（長文・資料） ≒ 30,000 トークン
3) 回数 × 重さ × 30日 × 単価 で、だいたいの月額が出る
     → 出力が長いほど跳ねる。だから Fable 5 は
       「難所だけ」に絞ると、ぐっと安くなる。

🔀2大モデル ハイブリッド運用レシピ30

Fable 5 と GPT-5.6 を「両方組み合わせて」使う実践レシピ30個。Claude Code の頭脳と手足、下書き→仕上げ、相互チェックなど。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 A. 下書き→仕上げ（速く形にして、要所を磨く）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. ブログ記事：GPT-5.6で骨子と下書き → Fable 5で論理の穴と言い回しを仕上げ
2. 企画書：GPT-5.6で叩き台を10案 → Fable 5で「通らない理由」を潰す
3. メール：GPT-5.6で3案 → Fable 5で相手の受け取り方をチェック
4. スライド構成：GPT-5.6で全体像 → Fable 5で抜け・順序を点検
5. 長文要約：Luna/Terraで一次要約 → Fable 5で重要点の取りこぼしを確認
6. LP文章：GPT-5.6で全ブロック → Fable 5でファーストビューだけ精度アップ

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 B. 頭脳と手足（Fable 5が考え、GPT-5.6が動く）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
7. 調査：Fable 5に「調べる観点」を設計させる → GPT-5.6で大量に収集・整形
8. 実装：Fable 5に仕様と受け入れ条件 → GPT-5.6／Codexが実装
9. 検収：GPT-5.6が作ったもの → Fable 5にレビューさせる
10. データ整形：Fable 5にルールを定義させる → Lunaで大量処理
11. 記事量産：Fable 5にテンプレを設計させる → GPT-5.6で本数を回す
12. リサーチ設計：Fable 5に「何を知るべきか」→ GPT-5.6で情報収集

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 C. 相互チェック（片方の答えを、もう片方に採点させる）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
13. 事実確認：GPT-5.6の答えをFable 5に「怪しい所は？」と検証させる
14. 逆採点：Fable 5の長い答えをGPT-5.6に「3行に要約して」と圧縮
15. 抜け探し：両方に同じ質問 → 差分を見て「片方だけが言った点」を拾う
16. 反論役：一方に主張、もう一方に「反対の立場で弱点を挙げて」
17. 訳の検証：GPT-5.6で翻訳 → Fable 5で誤訳・ニュアンスずれを点検
18. 数字チェック：計算や見積りは両方に出させて、食い違いを確認

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 D. 使い分けの型（場面で持ち替える）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
19. 難所のバグ：最初からFable 5（一発の精度）
20. 大量の仕分け：Luna一択（安く速く）
21. 日々の相談：Terraで十分、深い判断だけSol/Fable 5
22. 自律で長時間：GPT-5.6 Sol（エージェント）に任せる
23. 企画の壁打ち：Fable 5に質問役をさせる（盲点出し）
24. とにかく急ぎ：GPT-5.6（速さ優先）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 E. 日常運用（コストと精度を両立）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
25. まずTerra/GPT-5.6で回し、詰まった時だけFable 5に切り替える
26. effortは「難問=高、確認=低」でメリハリを付ける
27. Fable 5の高い枠は「計画」と「レビュー」だけに使う
28. プロンプトは短く（特にGPT-5.6は絞るほど品質が安定）
29. 同じ頼みを両方に投げ、自分の“合う方”の基準を作る
30. どちらの出力も、最後は必ず自分の目で確認してから使う

✓受け取り方・使い方

  1. 欲しいキットの「📋 コピー」を押す（またはダウンロード）
  2. Claude Code / Claude（Fable 5）／ ChatGPT（GPT-5.6）に貼り付けて使う
